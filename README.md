# Student Performance Data Analysis Dashboard

## Objective

This project is a basic data analysis dashboard built using Streamlit. It helps users explore and analyze student performance data interactively.

## Dataset

The dataset contains information about students, including:

- Student ID
- Age
- Gender
- GPA
- Study Time
- Absences
- Sports
- Music
- Tutoring
- Volunteering
- Grade Class

## Features

- Dataset Preview
- Number of Rows and Columns
- Data Types
- Statistical Summary
- Missing Values
- GPA Histogram
- Gender Bar Chart
- Clean and Interactive Dashboard

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Streamlit

## Installation

Install the required libraries:

```bash
pip install -r requirements.txt
```

## Run the Application

```bash
streamlit run app.py
```
